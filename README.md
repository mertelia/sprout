# Sprout

A clean, interactive web project built with Next.js and TypeScript. <br/>
Based on the course by developedbyed.

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Drizzle ORM
- PostgreSQL / Neon
- NextAuth
- Stripe
- Resend
- Algolia Search
- TipTap Editor
- React Hook Form + Zod
- Zustand
- Framer Motion
